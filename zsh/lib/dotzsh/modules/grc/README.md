grc
===

Setup grc[1] to colour output of Commands

Settings
--------

### Commands

To colourify a different set of commands, add the following to *zshrc*:

    zstyle ':dotzsh:module:grc' commands \
      'command1' \
      'command2' \
      'command3' \
      'command4'

  *This will over ride the default list of commands*
