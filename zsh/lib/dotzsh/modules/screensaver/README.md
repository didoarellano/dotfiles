Screensaver
===========

Trigger a *screensaver* if idle, currently only cmatrix

Settings
--------

### Timeout

To change the timeout from 300 seconds, add the following line to *zshrc*:

    zstyle ':dotzsh:module:screensaver' timeout '600'
